tomahawk-plugin-vsxu
====================

Visualizer plugin for Tomahawk

Depends:
--------

Latest Tomahawk Player.
Vovoid VSXu 0.4.0+ (http://github.com/vovoid/vsxu)


Compiling and running Tomahawk
------------------------------

Compile and Install:

    $ mkdir build && cd build
    $ cmake ..
    $ make
    $ make install

And live happily ever after :)
